# mvc-demo

## How to run
1. Clone the repo:
  ```bash
  git clone https://github.com/Inna275/mvc-demo.git
  ```
2. Navigate to the mvc-demo folder:
  ```bash
  cd mvc-demo
  ```
3. Navigate to the todo_list folder:
  ```bash
  cd todo_list
  ```
4. Run the app:
  ```bash
  go run .
  ```
5. Open your browser and go to
  ```bash
  http://localhost:8080
  ```
